## Nest Doc

[Nest](https://docs.nestjs.com) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# watch debug
$ npm run start:debug

# production mode
$ npm run start:prod
```

## Typeorm migration

```bash
# create file
$ npm run migration:create

# run
$ npm run migration:run

# create entity
$ npm run entity:create
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
